Primo terminale:
cd Desktop/progetto-voto
npx hardhat node


Secondo terminale:
cd Desktop/progetto-voto/backend
npm run clean:all
node server.js

Terzo terminale:
cd Desktop/progetto-voto
npx hardhat run scripts/deploy.ts --network localhost

cd frontend
npm run dev

-----------------------------------------------------

Per avviare il progetto utilizzare i file .bat, 
start_project.bat -> avvia il progetto
reset-start_project -> resetta e avvia il progetto
create-election.bat -> crea elezioni di test
